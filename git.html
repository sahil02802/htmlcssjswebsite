Git Commands

1. Git Clone

Download a repository from GitHub to your local machine.

git clone <repo-url>

2. Git Init

Initialize a new Git repository in your current directory.

git init

3. Git Add

Stage changes for the next commit.

git add .

4. Git Commit

Save your staged changes to the repository history.

git commit -m "message"

5. Git Push

Upload your local commits to the remote repository.

git push origin main

6. Git Pull

Fetch and merge changes from the remote repository.

git pull origin main

7. Git Fetch

Download changes from the remote but do not merge them.

git fetch origin

8. Git Branch

Create a new branch for development.

git branch new-branch

9. Git Merge

Merge a branch into the current branch.

git merge new-branch
